## вЪ° ACPF Algorithm Explained (For Your Presentation)

```
ACPF вАФ Adaptive Cognitive-Physical Fusion

Inputs:
  Physical:  Form Score, Range of Motion, Smoothness, Rep Count, Joint Angle
  Cognitive: Focus (Gaze), Emotion Score, Stress, Fatigue, Breathing BPM, Motivation

Pipeline per frame:
  1. Load base weights (exercise-type specific)
  2. Dynamic weight adjustment based on threshold rules:
     - High Fatigue (>70%)  вЖТ boost fatigue weight, reduce ROM weight
     - High Stress  (>65%)  вЖТ boost stress weight, boost focus weight
     - Poor Form    (<55%)  вЖТ boost form weight for injury prevention
     - Low Focus    (<45%)  вЖТ boost focus weight
     - Abnormal BPM         вЖТ boost breathing weight
  3. Normalise weights вЖТ sum = 1.0
  4. Compute Physical Score + Cognitive Score (weighted)
  5. Fuse вЖТ Raw Wellness Score
  6. EMA Temporal Smoothing (ќ±=0.3) вЖТ Smooth Wellness
  7. Compute ACPF Score = Wellness*0.5 + Motivation*0.3 + Form*0.2
  8. Risk Assessment вЖТ SAFE / CAUTION / STOP
  9. Log to session buffer
  10. Per-rep snapshot when rep count increments

Key Innovation:
  Unlike static weighted fusion systems, ACPF dynamically adapts
  its 7-dimensional attention mechanism at every frame, prioritising
  the most safety-critical signal in real time.
```

## рЯЫ†пЄП Troubleshooting

| Problem | Solution |
|---------|----------|
| `emotion_model.h5` not found | Run `python train_models/train_emotion.py` |
| `eye_state_cnn_model_finetuned.keras` not found | Run `python train_models/train_gaze.py` |
| YOLO download fails | Check internet; or manually place `yolov8n-pose.pt` in `models/` |
| Cognitive panel not appearing | Check browser console for JS errors; make sure Flask is running |
| Dashboard button missing | Complete at least one rep before stopping the workout |
| Camera permission denied | Allow camera in browser settings |
| Breathing shows `-- BPM` for first 10s | Normal вАФ needs 8-10 seconds of shoulder data to calibrate |
| TensorFlow not found | `pip install tensorflow` or `pip install tensorflow-cpu` |
